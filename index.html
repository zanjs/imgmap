主要快捷键

F5 — 重新加载页面，并再次显示加载图像的形式

S — 保存地图 的 PARAMS 到 本地

绘图模式 (长方形 / 圆形 / 多边形)

ENTER — 停止多边形绘图（或单击第一个帮手）

ESC — 取消新区绘图

SHIFT — 在绘制多边形断点结束绘制下一条边

编辑模式

DELETE — 删除选定区域

ESC — 取消选定区域的编辑

SHIFT — 编辑并保存为矩形比例

I — 选定区域的编辑属性（或一个地区DBL点击）

CTRL + C — 拷贝所选区域

↑ — 向上移动所选区域

↓ — 向下移动所选区域

← — 向左移动所选区域

→ — 向右移动所选区域
